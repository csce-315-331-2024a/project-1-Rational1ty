Pseudo-3D Game Engine

A pseudo-3D (also known as 2.5D) game engine created with Java that uses a technique called ray casting to make the game appear 3D while the underlying logic is 2D. This technique was used for many of the earliest "3D" games, such as Wolfenstein 3D. Since the engine only works in 2D, these types of games can run on much weaker devices than modern 3D games, especially those without a GPU.

I really enjoy playing around with this project since I can add whatever textures, sprites, sounds, and NPCs that I want. In the future, I'd like to use this engine to make an actual game similar in style to early games like Doom.

Nearest Plane App

This is a simple yet interesting web app that uses the powerful OpenSky API to collect and process all sorts of interesting data about aircraft. It can tell you the closest plane to your current location, or any location in the world. It also shows you tons of cool data about each of the planes it detects, along with each flight's callsign and airline.

Conway's Game of Life

Conway's game of life is one of my favorite games to create in any programming language. This is a simple implementation written in Java, but it has several nice features that make it easy to experiment with new patterns and designs. It's a great way to waste time——just start the game and become mesmerized by all the spirals, flowers, gliders, and other cool visuals!
